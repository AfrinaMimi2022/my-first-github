Create a new repository on the command line

  1. echo "# my-first-github" >> README.md
  2. git init
  3. git add README.md
  4. git commit -m "first commit"
  5. git branch -M main
  6. git remote add origin https://github.com/AfrinaMimi2022/my-first-github.git
  7. git push -u origin main
